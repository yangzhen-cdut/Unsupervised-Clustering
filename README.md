## Requirements

The recommended requirements for TS2Vec are specified as follows:
* Python 3.7
* torch==1.8.1
* scipy==1.7.3
* numpy==1.21.6
* pandas==1.3.5
* scikit_learn==0.24.2
* matplotlib==3.5.2
* Bottleneck==1.3.4
* seaborn==0.11.2
The dependencies can be installed by:
```bash
pip install -r requirements.txt
```
